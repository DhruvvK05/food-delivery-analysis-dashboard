# Food Delivery Analysis Dashboard

A dynamic and interactive Power BI dashboard built to analyze food delivery business performance—focusing on order trends, delivery efficiency, restaurant performance, customer behavior, and operational insights.

---

## 📖 Short Description / Purpose

The Food Delivery Analysis Dashboard is a visually engaging and analytical Power BI report designed to help users explore and understand operational and business performance in a food delivery environment.

The dashboard focuses on highlighting delivery trends, restaurant performance, customer ratings, order volume, revenue patterns, and city-level operations.

This tool is intended for:
- Business Analysts
- Operations Teams
- Restaurant Partners
- Delivery Platforms
- Data-Driven Decision Makers

The goal is to deliver actionable insights that improve delivery operations, customer experience, and business performance.

---

## 🛠️ Tech Stack

The dashboard was built using the following tools and technologies:

- 📊 **Power BI Desktop** – Main data visualization platform used for report creation and dashboard development.
- 📂 **Power Query** – Data transformation and cleaning layer for preprocessing and reshaping data.
- 🧠 **DAX (Data Analysis Expressions)** – Used for calculated measures, KPIs, and dynamic analysis.
- 📝 **Data Modeling** – Relationships and calculated tables created to enable filtering and aggregation.
- 📁 **File Formats** – `.pbix`, `.csv`, `.png`

---

## 📂 Data Source

**Source:** Food Delivery Dataset

The dataset contains operational and transactional food delivery data including:

- Orders
- Restaurants
- Customers
- Delivery Times
- Ratings
- Cities
- Revenue
- Delivery Status
- Order Volume
- Cuisine Categories

The dataset was cleaned and transformed using Power Query before integration into Power BI.

# 🚀 Features / Highlights

---

## • Business Problem

Food delivery platforms handle large volumes of operational and customer data daily, making it difficult to quickly identify performance bottlenecks and business trends using raw datasets alone.

Key questions such as:

- Which restaurants generate the highest revenue?
- Which cities contribute the most orders?
- How efficient are delivery operations?
- Which cuisines are most popular?
- What factors influence customer ratings?
- How do delivery times impact customer experience?

... are difficult to answer efficiently without an interactive analytics dashboard.

---

## • Goal of the Dashboard

To deliver an interactive visual analytics tool that:

- Enables users to explore food delivery business performance.
- Supports operational and strategic decision-making.
- Identifies high-performing restaurants and cities.
- Tracks delivery efficiency and customer satisfaction.
- Provides insights into customer ordering behavior and revenue trends.
- Enables interactive filtering and drilldown analysis.

- Total Orders

### Key Visuals

- Cuisine Category Analysis
- Customer Rating Distribution
